# Naive CPU TODO / 未实现列表

## 指令与功能缺口
- **Load 扩展**：仅实现 LW，对 LB/LH/LBU/LHU 未做符号/零扩展，未对齐策略仅日志。
- **Store/访存异常**：对齐/越界仅日志，不会触发 trap/halt。
- **FENCE / 非法指令**：未解码或未处理，非法 opcode/funct 不触发异常。
- **系统指令**：ECALL/EBREAK 直接 `finish()`，未区分 cause/未提供 trap 状态。
- **RV32M/CSR 等扩展**：未实现。

## 流水/控制
- **取指循环**：Driver 只启动一次，WB/其他阶段未触发下一条 fetch，无法连续执行。
- **WB 控制**：无显式 `rd_en`，trap/异常时未屏蔽写回；rd_used 推送方式简化。
- **冒险/转发**：无数据/控制冒险处理，默认一条指令串行走完整链路。
- **分支/跳转对齐异常**：仅日志，不会停止或记录 trap。

## 访存路径
- **dcache 接口**：EX 直接触发 dcache，MA 读取 `dout`；缺乏握手/等待，未处理 SRAM 错误。
- **Load 数据处理**：仅 32bit，未实现按 `funct3` 宽度和符号扩展。

## 其他
- **命名警告**：生成的 Rust 代码有大量 snake_case 警告，暂未处理。
- **测试缺口**：仅有解码单测，缺少执行/访存/分支等指令序列测试及 trap 测试。
